Question 1 (20 marks)

This question assesses the Introduction and Section 1 of Unit 1, and the topic of Big-O notation as dealt with in Section 4 of Unit 2.

In order to answer this question, you will need to use the Name search [Tip: hold Ctrl and click a link to open it in a new tab. (Hide tip)]   online application.

The application allows the user to search from among the 90 most popular girls’ names in 2012 (England and Wales), and obtain the popularity rank of a chosen name, as well as the total number of babies given that name in 2012.

Before attempting to answer the following questions, we suggest that you go to the Name search application and try entering some names in the ‘Name:’ input box. You should see that the list of names is repeatedly partitioned until either the name is found, or the list of names is empty, in which case the result ‘Not found’ is returned in the ‘Rank’ and ‘Total babies’ fields.

The search algorithm is the same as Algorithm 1.2 in the discussion of Exploratory activity 1.5 in Section 1.1 of Unit 1.

  • a. 

    • i.Which name does the algorithm find with the fewest iterations?
    • ii.How many iterations does the algorithm take for this name?

    (2 marks)

  • b. 

    • i.How many iterations does the algorithm take to find the name ‘Imogen’.
    • ii.Describe, in general terms, how a single iteration of the algorithm operates.
    • iii.Describe each of the individual iterations when ‘Imogen’ is the search term.
    • iv.Is there a name that requires more iterations than ‘Imogen’? If so, give that name; otherwise, explain why there is no such name.

    (5 marks)

  • c.Explain:
    • i.how the data has been structured
    • ii.why this structure allows the algorithm to search for the rank of a chosen name more efficiently.

    (3 marks)

  • d.Consider the following the decision problem:

    Given a name, N, is N in the list of the 90 most popular girls’ names in 2012?

    • i.Explain why this problem is a decision problem.
    • ii.Modify Algorithm 1.2 to implement an effective procedure which will solve every instance of the decision problem.

      Copy Algorithm 1.2 into your Solution Document, and make the required changes.

    (6 marks)

  • e. 

    • i.What is the smallest number of iterations your algorithm might need to solve an instance of the decision problem?
    • ii.What would the largest number of iterations have been if there had been 180 names in the list? Explain your answer.
    • iii.Using Big-O notation, what is the number of iterations needed by your algorithm if there are n names in the list, where n is a positive whole number? Explain your answer.

    (4 marks)
